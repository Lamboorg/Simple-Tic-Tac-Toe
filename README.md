# Simple Tic-Tac-Toe 🎮

A console-based Tic-Tac-Toe game written in Kotlin, where two players take turns on a 3×3 grid until someone wins or it's a draw.

Built as part of the [Hyperskill Kotlin track](https://hyperskill.org).

## How it works

```
 X | O | X
-----------
   | X | O
-----------
 O |   | X
```

Players input coordinates (row, column) to place their mark. The game checks for a win after every move, and announces the winner — or a draw if the board fills up.

## Tech

- 🟣 Kotlin
- 🛠️ Gradle

## How to run

1. Clone the repo
2. Open in IntelliJ IDEA
3. Run the `Main.kt` file
